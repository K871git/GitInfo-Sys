# modify files 
git status 

1) untracked    : new files in repo that haven't tracked yet 
    git add <filename> or (.) for all changes
2) modified     : changed files
    git status  : this will show the changed (red color) and staged(green) files 
    git diff    : to see the change
3) staged       : files or changes are ready to be commited 
4) unmodified   : No changes since last commit :: git status shows clean
